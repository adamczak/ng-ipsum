ng-ipsum

AngularJS randomness

(like above)

Words

Ipsum-y text.

As a directive

Add the ipsum attribute to an element.



<div ipsum="10w"></div>
<div ipsum="2s"></div>
<div ipsum="1p"></div>

Options

The value of the ipsum attribute defines the output. Format is number of items + type: \d+w|s|p. Options can be chained, i.e.: <span ipsum='2w1p3s'/> for 2 words, 1 paragraph and 3 sentences.

Value Meaning
10w 10 words, no punctuation
2s 2 sentences. First letter of each capatalized, period at end.
1p 1 paragraph. Will be wrapped in <p></p> tags

Names

Ipsum-y names.

As a directive

Add the ipsum-name attribute to an element.



<div ipsum-name="mfml"></div>
<div ipsum-name="flmf"></div>
<div ipsum-name="rf"></div>

Options

The value of the ipsum-name attribute defines the output. Format is sex + format: [fmr][fml]+.

Option Values
Sex m = male
f = female
r = randomly pick m or f
Format f = first
m = middle initial
l = last
One or more is required, and only 1 of each can be specified.
Output will be in order provided, i.e.: <span ipsum-name='fmlf' /> will output a female name in the format MI Last First
