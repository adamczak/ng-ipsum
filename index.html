angular-ipsum

AngularJS randomness

Getting started

bower install angular-ipsum or download from here.

Then include it in the list of dependencies for your application:

var app = angular.module("app", ["ipsum"]);

Words

Ipsum-y text.

As a directive

Add the ipsum attribute to an element.



<div ipsum="10w"></div>
<div ipsum="2s"></div>
<div ipsum="1p"></div>

Options

The value of the ipsum attribute defines the output. Format is number of items + type: \d+w|s|p. Options can be chained, i.e.: <span ipsum='2w1p3s'/> for 2 words, 1 paragraph and 3 sentences.

Value Meaning
10w 10 words, no punctuation
2s 2 sentences. First letter of each capatalized, period at end.
1p 1 paragraph. Will be wrapped in <p></p> tags

Names

Ipsum-y names.

As a directive

Add the ipsum-name attribute to an element.



<div ipsum-name="mfml"></div>
<div ipsum-name="flmf"></div>
<div ipsum-name="rf"></div>

Options

The value of the ipsum-name attribute defines the output. Format is sex + format: [fmr][fml]+.

Option Values
Sex m = male
f = female
r = randomly pick m or f
Format f = first
m = middle initial
l = last
One or more is required, and only 1 of each can be specified.
Output will be in order provided, i.e.: <span ipsum-name='fmlf' /> will output a female name in the format MI Last First

As a service

Inject the ipsumService into your controller/service/directive

Operation
ipsumService.paragraphs(count) Get a random string of sentences in paragraph format (wrapped in <p> tags)
ipsumService.sentences(count) Get a random string of words in sentence format
ipsumService.words(count) Get a random string of words
ipsumService.randomFemale() Get a random female first name
ipsumService.randomMale() Get a random male first name
ipsumService.randomFirst() Get a random first name
ipsumService.randomLast() Get a random last name
ipsumService.randomMi() Get a random middle initial (a letter)
ipsumService.randomName() Get a random 'person' object - {first: '', mi: '', last: '', full: ''}
ipsumService.randomItem() Get a item from a provided list

Example

app.controller('peopleController', function($scope, ipsumService) { 
    $scope.person = ipsumService.randomName('r');
    $scope.people = [];

    for(var i = 0; i < 10; i++)
    {
        var person = ipsumService.randomName('r');
        person.email = person.first + '.' + person.last + '@' + ipsumService.words(1) + ipsumService.randomItem(['.net','.org','.com','.biz'])
        $scope.people.push(person);
    }
});
